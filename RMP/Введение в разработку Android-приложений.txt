	История Андроид
	Android - открытая операционная система для мобильных телефонов, смартфонов, коммуникаторов, планшетный пк, эл. книг, цифр проигрывателей и тд. основанная на ядре линукс.
	Изначально разрабатывалась компанией Android Inc, которую затем в 2005 купила Google. Впоследствии Google инициировала создание бизнес альянса Open Handest Alliance, который занимается поддержкой и дальнейшим разрабатыванием платформы.
	
	Инструментарий разработчика
	Разработка Andoid-приложений, как правило, осуществляется на языке JAVA, который ООП. Программы на JAVA, транслируется в байт-код, выполняемый виртуальной машиной JAVA, которая обрабатывает байтовый код и передаёт инструкции к оборудованию, как интерпретатор. Достоинство подобного способа выполнения программ заключается в полной независимости байт-кода от операционной системы и оборудования, что позволяет выполнять Java-приложения на любом устройстве, для которого существует соответствующая виртуальная машина. Другой важной особенностью Java является гибкая система безопасности, благодаря тому что исполнение программы полностью контролируется виртуальной машиной. Любые операции, которые превышают установленные полномочия программы вызывают немедленное прерывание.
	
	Для разработчика Android-Приложений, в первую очередь, необхадимо установить Java Devolpment Kit (JDK). JDK - это бесплатно распространяемый комлект разработчика приложений на языке Java, включающий в себя компилятор Java, стандартные библеотеки классов Java, документацию, различные утилиты и исполнительную систему Java RunTime Enviroment (JRE).
	В состав JDK не входит интегрированная среда разработки, поэтому после JDK, следует установить IDE. Существует несколько популярных сред разработки, но в данном курсе будет использована Android Studio
	Android Software Devolpment Kit (SDK) содержит множество интсрументов и утилит для создания и тестирования приложений. к примеру с помощью SDK Manager можно установить Android Api для любой версии, а также проверить репозитории на наличие доступных, но ещё не установленных пакетов и архивов.

	Приложения 
	Android поставляется с набором основных приложений: Календарь, карты, браузер, менеджер контактов и т.д. Все пересчисленные написаны на Java.
	
	Платформа приложений.
	Предоставляя открытость, открытую платформу разрабочтки, дается возможность создавать гибкие и иновационные приложения. Могут использовать аппаратные возможности устройства, получать информацию о местоположении, выполнять задачи в фоновом режиме, устанавливать оповещения и многое другое. разработчики имеют полный доступ к тем же API что в основных приложениях.
	Архитектура разработана с целью цпрощения повторного использования компонентов. Любое приложение может опубликковать свои возможности и любое другое может исползовать эти возможности. Этот механизм позволяет заменять стандартные компоненты на пользоваетльские.
	
	Библеотеки
	Android включает в себя набор C/C++ библеотек, используемых различными компонентами системы. Эти возможности доступны разработчикам в контексте применения Android Application Framework. некоторые основные библеотеки пречисленны ниже.
	- Медиобиблеотеки. Поддержка воспроизведения и записи популярных аудио и видео форматов и изображений.
	- Surface Manager - управляет доступом к подсисиеме отображения 2d и 3D 
	- LibWebCore - современный веб - движок, на котором построен браузер android.
	- SGL - основной графический движок 2D 
	- 3D библеотеки - реализованы на основе OpenGL. Библеотеки используют либо аппаратное 3D ускорение либо вклчает программно.
	- FreeType - поддержка растровых и векторных шрифтов.
	- SQLite - механизм базы данных, доступной для всех приложений.
	
	ВЫполняемая среда Android
	Оно включает в сея набор основных библеотек, которые обеспечивают большинство функций, доступных в библеотеках Java. Каждое приложение Andriod своём собственном процессе со своим экземпляром виртуальной машины Dalvik, была написана так, что устройство мжет работать эффективно с несколькими вирутальными машинами одновременно.
	DAlvik проектиировалась под систему Android. оПТИМИЗИРОВАНА ДЛЯ низкого потребления памяти и работы на моб аппаратном обеспечении, использует собственный байт-код. Приложения Андроид переводятся компиляторо в специальный машинно независимой низкоуровненвый код, а именно Dalvik интерпертирует и ыполняет такую программу при выполнении на платформе, кроме того с дополнений может переводить на Java.
	
	Ядро линукс
	Андроид основан на Linux версии 2.6 с основными системами службами - безопасностью, управлением памятью, управлением процессами и моделью драйверов.
	
	Обзор Java-Интерпритаторов.
	Основана для прикладного програмиста Android - это набор интерфейсов на языке Java. В основе набора - пакеты входящие в стандарт языка Java, такие java.util, java.lang, java.io. Также примыкают , котрые не входят в стандарт языка: java.net, java.xml и др.
	Пакеты andrioid view и anroid.widget отвечают за графичесакий интерфейс пользователя. Они содержат набор встроенных виджетов, таких как кнопки и поля ввода, разметки для располжоения виджетов на экране. С их помощбю можно создать простейшее приложение андроеда.
	Для работы с примитивами рисования и графическими файлами предназначен пакет android.graphics, а с помощью android.animation можно создавать простую анимацию.
	Пакет android.opengl предоставляет движок OpenGl Android.gesture добваляет поддержку управденяия жестами на  сенсорном экране, позволяет распозновать жесты и создавать новые.
	android.net включет стеки сетевых протоколов выского уровня, таких как Http и SIP. поддержку  Wifi. Android Webkit - популярный движок веб-браузера. android.bluetooth и android.nfc предоставляют стеки протоколов связи на коротких расстояних.
	для управления прикладными приложениями предназначен пает android.app. пакет Android.hardware позволяет обращятся к камере и датчикам, а пакет android.location предоставляет информацию о географических координатах. Пакет android.media отвечаает за кодирование звуковых и видео потоков, для мобильных устройств это до сех пор сложная задача, требующая качетсвенной оптимизации. Пакет android.database предоставляет доступ к баззам данных.
